# Ustab.uz

**Ustab.uz** is an online platform designed to help users find builders and purchase construction materials or compare their prices. The platform aims to simplify the process of finding skilled professionals and sourcing materials for construction projects.

## Features

- **Find Builders**: Search and connect with professional builders for various types of construction projects.
- **Compare Material Prices**: Find the best prices for construction materials and compare across different suppliers.
- **User-Friendly Interface**: Easy-to-use design that prioritizes usability and convenience over complex features.
- **Firebase Authentication**: Secure login and authentication for users via email/password or Google Sign-In.
- **Responsive Design**: Optimized for both mobile and desktop platforms for a seamless experience.

## Tech Stack

- **Frontend**: React.js
- **Backend**: Firebase
- **State Management**: React Context API (or use Redux if preferred)
- **Authentication**: Firebase Authentication (Email/Password and Google Sign-In)
- **Hosting**: Firebase Hosting (optional, depending on deployment)
- **UI**: Simple, clean UI with emphasis on user experience.
